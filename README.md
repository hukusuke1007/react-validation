## How to install

```sh
git clone 
cd imory_frontend
yarn

# run
yarn start
```

```sh
# test
yarn jest
```

## Development & Deploy

環境設定をする。

```sh
# 開発環境
./env_development.sh

# 本番環境
./env_production.sh
```

デプロイは次のコマンドを使用する。

```sh
./deploy_web.sh
```
